# Guide utilisateur — EduWeb Booking

## Se connecter
Rendez-vous sur `/login`, saisissez votre e-mail et votre mot de passe. En démonstration, utilisez
un compte de la liste (mot de passe `password123`) ou les boutons de connexion rapide.

## Réserver une ressource
1. Cliquez sur **Nouvelle réservation** (bouton vert, en haut ou dans la barre latérale).
2. **Catégorie** — choisissez le type de ressource (salle, matériel, véhicule…).
3. **Ressource** — sélectionnez la ressource souhaitée.
4. **Créneau** — choisissez la date et l'heure, puis cliquez sur **Vérifier la disponibilité**.
   - ✅ Vert : le créneau est libre.
   - 🟠 Orange : conflit détecté — une **alternative** vous est proposée.
5. **Motif** — décrivez l'objet et le type d'usage, indiquez l'effectif.
6. **Détails** — besoins particuliers, assistance technique, note pour le validateur.
7. **Confirmation** — vérifiez le récapitulatif et **soumettez la demande**.

Vous recevez un accusé. Selon la ressource, la demande est validée automatiquement ou passe
**en attente de validation**.

## Suivre mes réservations
**Mes réservations** liste vos demandes (à venir / historique) avec leur statut. Cliquez sur une
réservation pour voir le détail, le suivi (timeline) et les actions disponibles.

## Présence
Une fois la réservation **validée** et le créneau arrivé :
- **Je suis arrivé** → la réservation passe « en cours » ;
- **Activité terminée** → la réservation est clôturée.

## Annuler
Depuis le détail d'une réservation à venir, cliquez sur **Annuler la réservation**.

## Codes couleur des statuts
| Couleur | Signification |
|---|---|
| 🟢 Vert | Disponible / Validée |
| 🟠 Orange | En attente / Action requise |
| 🔴 Rouge | Indisponible / Refusée |
| 🟣 Violet | En cours / Module avancé |
